Introduction
============

Momonga is MQTT server written in Golang. MQTT has ability to provide M2M(Machine to machine), IoT(Internet of things) infrastructure easily.
Also, it is useful for just PUBSUB something.
Momonga started as a personal project which learning MQTT specification and how to write Go middleware.
